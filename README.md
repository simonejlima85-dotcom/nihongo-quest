# Simone no Nihongo Quest 🌸

Planner gamificado de japonês (Dark Edition) + Kanji Quest

## 🔗 Link online

**https://simonejlima85-dotcom.github.io/nihongo-quest/**

## Como instalar como App no celular (PWA)

### Android (Chrome)
1. Abra o link acima no **Chrome**
2. Toque no menu (três pontinhos ⋮)
3. Escolha **"Adicionar à tela inicial"** ou **"Instalar aplicativo"**

### iPhone (Safari)
1. Abra o link no **Safari**
2. Toque no botão **Compartilhar** (quadrado com seta)
3. Role e escolha **"Adicionar à Tela de Início"**

## Se o site estiver quebrado

1. Espere 1-2 minutos e atualize a página (puxe pra baixo ou F5)
2. Ou baixe o arquivo HTML completo e abra direto no navegador

## Arquivos

- `index.html` → App principal
- `manifest.json` + `sw.js` → suporte a PWA

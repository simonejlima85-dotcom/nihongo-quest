# Simone no Nihongo Quest 🌸

Interactive Japanese learning planner + Kanji Quest (Dark Edition)

## Live App

**[Abrir o Nihongo Quest](https://simonejlima85-dotcom.github.io/nihongo-quest/)**

## Features

- Daily missions (Yoga / Curso / Almoço)
- Boss battle every Friday with Makoto
- 100+ real kanjis from your conversations
- Flashcards + Quiz
- XP, Streak, Levels and Achievements
- Everything saved in browser (localStorage)
- Dark theme

Made with love for Simone ❤️
